專題簡介

惡意流量模擬與防禦測試平台

本專題旨在開發一個網站平台，提供輕量化測試 agent，讓企業能模擬各種惡意攻擊行為， 驗證其 IPS/IDS 防禦能力。透過惡意程式樣本分析與特徵提取，平台可生成詳細測試報告， 協助企業即時發現防禦弱點並調整策略，避免防禦失效與資安風險。

Agent · Windows

Windows 測試 Agent

以 WinINet 與 HTTP 傳輸模擬惡意流量，驗證 IPS/IDS 攔截狀況與規則有效性。

Agent · Linux

Linux 測試 Agent

這個版本是專門給 Linux 使用者的測試工具。 它會模擬出一些「看起來像病毒攻擊」的網路行為，並把這些資料送回伺服器。 透過這樣的測試，你可以知道自己的防禦系統（IPS/IDS）能不能擋下， 如果擋不下，就能立刻發現弱點並加以改進。
